# Session Close Memory Prompt

Convert session outcomes into durable memory and clean temporary noise.

## Checklist

1. Add directives for new standing rules.
2. Add architecture/implementation notes for key decisions.
3. Write one concise working-memory summary entry.
4. Prune stale working entries with `notepad_prune`.
5. Confirm final notepad stats with `notepad_stats`.

## Required Output

- `directives_added`
- `notes_added`
- `working_summary_added`
- `prune_result`
- `stats`
