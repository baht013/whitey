Whitey is a lightweight local development orchestration tool for AI coding workflows.
It gives developers one central place to manage agents, skills, hooks, memory, policies, MCP tools, and provider routing.
Instead of using one AI assistant for everything, Whitey can delegate deep planning to stronger models, small operational tasks to cheaper/faster models, and run validation hooks after code changes.
The goal is to make local AI-assisted development faster, safer, and more structured without becoming a heavy platform.
